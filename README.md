# Store Operations Intelligence

Empowering AI Agent Publishing with Intelligent Operations

## Quick Start

### Option 1: Python HTTP Server (Recommended - No dependencies)

```bash
# Python 3
python3 server.py

# Or Python 2 (if Python 3 not available)
python server.py
```

The server will start on `http://localhost:8000` and automatically open in your browser.

### Option 2: Node.js HTTP Server

First, install dependencies:
```bash
npm install
```

Then start the server:
```bash
npm start
```

For development with no caching:
```bash
npm run dev
```

### Option 3: Simple Python One-liner

```bash
# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Then open `http://localhost:8000` in your browser.

### Option 4: PHP Built-in Server

```bash
php -S localhost:8000
```

## Accessing the Application

Once the server is running, open your browser and navigate to:
- **Main URL**: `http://localhost:8000/`
- **Direct HTML**: `http://localhost:8000/index.html`

## Features

- **Business Leader Dashboard**: Strategic insights and operational metrics
- **Engineering Team Dashboard**: Technical monitoring and alerts
- **Product Team Dashboard**: Comprehensive analytics and metrics

## Development

The application is a static site with:
- `index.html` - Main application file
- `script.js` - JavaScript functionality
- `styles.css` - Styling

## Stopping the Server

Press `Ctrl+C` in the terminal where the server is running.
